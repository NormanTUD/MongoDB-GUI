# MongoGUI

A very simple GUI for viewing, deleting, adding and updating MongoDB-Databases with PHP without needing to memorize MongoDB's kinda quirky language.

# How to chose DB

```console
echo $DBNAME > dbname
echo $COLLNAME > collname
```
